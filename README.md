# Student Profile Management System

## Overview

The Student Profile Management System is a desktop-based application developed using Python, Tkinter, and MySQL. The system is designed to efficiently manage student records, academic information, attendance, fee details, and semester results through a user-friendly graphical interface.

## Features

* Secure Login System
* Student Record Management

  * Add Student
  * Update Student
  * Delete Student
  * View Student Records
* Student Search Functionality
* Attendance Management
* Fee Management
* Semester Result Tracking (Semester 1–8)
* Backlog Management
* Guardian Information Management
* Student Ranking System
* Export Data to Excel
* Data Validation for User Inputs
* MySQL Database Integration
* Attractive Tkinter GUI

## Student Information Stored

* Student Name
* ERP ID
* Class Name
* Email ID
* Mobile Number
* Blood Group
* Date of Birth
* Address
* Current Semester
* Enrolled Courses
* Certificates
* Fee Status
* Semester Results
* Backlogs
* Father's Details
* Mother's Details
* Guardian Details
* Attendance Information

## Installation Guide

### 1. Clone Repository

git clone https://github.com/Ayushkumar396/Student-Profile-Management-System.git

### 2. Navigate to Project Folder

cd Student-Profile-Management-System

### 3. Install Required Libraries

pip install pandas openpyxl mysql-connector-python

### 4. Configure MySQL Database

Create the required database and tables in MySQL.

Update the database credentials in the Python source code:

host = "localhost"
user = "root"
password = "your_password"

### 5. Run Application

python main.py
